# 알뜰폰 매니저 UI 리디자인 — v7.0

**날짜:** 2026-05-07  
**범위:** UI/UX 개선 (기능 추가 없음, 기존 로직 유지)

---

## 목표

v6.2의 기능은 그대로 유지하면서 시각적 완성도와 사용성을 높인다.  
테마: **밝은 클린** — 화이트 배경 + 블루(#3b82f6) / 시안(#06b6d4) 포인트.

---

## 변경 범위

### A. 헤더 슬림화

**현재:** 앱 이름 + 강제 업데이트 버튼 + 그 아래 PIN 섹션 카드가 크게 자리 잡음.

**변경:**
- 헤더 1줄 압축: `📱 알뜰폰 관리 v7.0` + 오른쪽 아이콘 버튼 2개 (☁️ 클라우드, 🔄 새로고침)
- PIN 입력은 ☁️ 버튼 클릭 시 슬라이드 다운 패널로 노출 (기본 숨김)
- 앱 설치 버튼은 헤더 우측에 작은 배지로 유지

### B. 메인 카드 개선

**현재:** 보라/인디고 그라디언트 고정, D-day 숫자가 작고 어두움.

**변경:**
- 기본 색상: 블루→시안 그라디언트 (`#3b82f6` → `#06b6d4`)
- D-day 숫자 폰트 크기 확대 (현재 2.5rem → 3rem)
- 상태별 카드 색상 자동 변환:
  - **정상** (15일+): 블루/시안
  - **임박** (1–14일): 오렌지/앰버 (`#f59e0b` → `#f97316`)
  - **만료** (D+): 레드 (`#ef4444` → `#dc2626`)
- 정보 행을 3열 그리드 칩으로 변경: 가입일 / 종료일 / 월 요금

### C. 입력 폼 UX 개선

**현재:** 통신망을 `<select>`로 선택 후 MVNO `<select>` 활성화.

**변경:**
- 통신망 선택을 탭 버튼 3개로 변경 (SKT / KT / LG U+), 클릭 시 즉시 MVNO 목록 갱신
- MVNO는 기존 `<select>` 유지 (목록이 많아서)
- 가입일 입력 옆에 **"오늘"** 퀵 버튼 추가
- 폼 섹션 구분: 요금제 정보 / 기간·비용 정보 / 부가 정보 3구역으로 구분선 + 레이블 추가

### D. 할인 기간 진행 바

**현재:** 없음.

**변경:**
- 메인 카드 D-day 아래에 Progress bar 추가
- 계산: `(오늘 - 시작일) / (종료일 - 시작일) * 100`%
- 바 색상은 카드 상태와 연동 (정상: 흰색/반투명, 임박: 오렌지, 만료: 빨간 100%)
- 바 아래 `시작일 ~ 종료일` 텍스트 표시
